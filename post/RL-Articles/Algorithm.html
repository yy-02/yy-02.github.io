DQN & DDQN

DQN

DQN is an algorithm that combine deep learning with -learning. In traditional -learning, we usually use a matrix to save the value of for each state-action combination(Or using function approximation in large questions).

In DQN we have :

  • A -estimation network
  • A -target network
  • An experience pool

In a learning period, the weight of -target network is fixed and -estimation network is updated by the value of state-action combination in -target network. At the end of this period, the weight of -target network is updated by -estimation network.

Disadvantage : In DQN the value of will be overestimated, because -esitimation network is updated by -target network.

DDQN

To solve the over estimation problem above, DDQN is proposed. The difference between DQN and DDQN is that, in DDQN action is selected by its value in -estimation network and then -target network will use this action to update the value of .

Incremental discretization of the states and actions space

In this article they presented the main ideas of incremental discretization of the state space and action space developed in Carlucho et al.(2017). Briefly speaking, at first they have a fairly coarse discretization of the action space , then if action is selected it will be divided into two parts (or we can say it is a splitting process). As the discretization process going on, actions which have been chosen more will have higher discretization level.
The final result represents the discretization of action space .
